SimpleXDT99IDE
is a simple application which implements a small IDE that shows you how to use the XDTools99.framework.

This application is part of the xdt99-warpper project. All source of the xdt99-wrapper is released under the GNU Lesser General Public License, Version 2.1 (GNU LGPLv2.1).

This code uses parts of the TI 99 Cross-Development Tools project (xdt99, version 1.6.0) by Ralph Benzinger (aka endlos99) as a library through the Python C-API.

For more Information read the README.md or visit the source repository at github.

Enjoy!
